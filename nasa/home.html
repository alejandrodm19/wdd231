Site Name

NASA Explorations

The name represents the website's purpose of showcasing NASA's latest space missions, images, and discoveries using NASA's public API.

Site Purpose

The website provides an interactive experience for space enthusiasts by fetching and displaying real-time data from NASA’s API. Users can explore space images, mission updates, and astronomical data in a structured and engaging format.

Scenarios

  • What is the latest image from NASA’s Astronomy Picture of the Day (APOD)?
  • How can I see details of NASA's latest space missions?
  • Where can I find images from Mars rovers?
  • What are the latest astronomical discoveries?

Color Schema

Primary Colors:

  • #0B3D91 (Deep Space Blue) - Used for headers and primary elements
  • #FFCC00 (Solar Gold) - Used for highlights and accents
  • #FFFFFF (White) - Used for text and backgrounds
  • #1A1A1A (Black) - Used for footers and contrasting elements

Typography

  • Montserrat - Used for headings and titles
  • Open Sans - Used for body text
